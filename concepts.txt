json.parse::

How to Use JSON.parse()

The basic syntax for using JSON.parse() is straightforward:


javascript
const obj = JSON.parse(text);

text: The required first argument is the JSON-formatted string you want to convert.
obj: The return value is the resulting JavaScript object or array. 

Examples
Parsing a JSON Object String

{
javascript
const jsonString = '{"name":"John", "age":30, "city":"New York"}';

// Use JSON.parse() to convert the string to a JavaScript object
const userObject = JSON.parse(jsonString);

// Now you can access properties using dot notation
console.log(userObject.name); // Output: John
console.log(userObject.age);  // Output: 30        

}


Parsing a JSON Array String
If the JSON data represents an array, JSON.parse() will return a JavaScript array: 
{
javascript
const jsonArrayString = '["Ford", "BMW", "Audi"]';

// Use JSON.parse() to convert the string to a JavaScript array
const carArray = JSON.parse(jsonArrayString);

// Now you can access elements using array indexing
console.log(carArray[0]); // Output: Ford

}

1️⃣ find()
Purpose: Multiple documents ko search karne ke liye
Return: Array of documents (empty array bhi ho sakta hai agar match na ho)
Filter: Query object dekar filter kar sakte ho

Syntax:
Model.find(filter, projection, options)

filter → kaunse documents chahiye ({ name: "Apple" })
projection → kaunse fields chahiye ({ name: 1, price: 1 })
options → sorting, limiting, etc

Notes:
Agar koi match na ho → [] (empty array) return
Multiple results ko handle karne ke liye use hota hai

++++++++++++
2️⃣ findById()
Purpose: Single document by _id (MongoDB ObjectId)
Return: Single document object ya null agar na mile
Shortcut: _id ke liye query likhne ka shortcut

Syntax:
Model.findById(id, projection, options)

id → document ka _id (string ya ObjectId)
projection → kaunse fields chahiye
options → populate, lean, etc

Notes:
Agar document exist na kare → null return
findById() internally findOne({ _id: id }) ka shortcut hai
++++++++++++++

| Feature           | `find()`                          | `findById()`                   |
| ----------------- | --------------------------------- | ------------------------------ |
| **Use case**      | Multiple documents search         | Single document by `_id`       |
| **Return type**   | Array (`[]` if no match)          | Single document object or null |
| **Filter**        | Any query object                  | Only `_id`                     |
| **Example**       | `Product.find({category:"Book"})` | `Product.findById("64f3...")`  |
| **Multiple docs** | Yes                               | No                             |
